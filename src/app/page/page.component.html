The monitor is a real-time device that is watching people, analyzing incidents or systems for changes, and raising red flags as soon as they happen. The system's prominent role is to send out alerts when certain events occur based on sensor measurements.

This gathers sensor data from various sensors, including temperature sensors, humidity sensors, pressure sensors, etc. When an alarm occurs, the user is notified via email, SMS, or phone calls. The user can choose which notification channels they want to receive notifications from.

The monitor shows the consumer the sensor readings for each sensor as well as the sensor's status. The sensor readings are displayed as graphs for each sensor, and the user can access the user interface to view past notifications for each sensor.
